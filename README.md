# CRM Leads Management System (LMS)

A Leads Management System (LMS) is a tool that helps businesses capture, track, and manage potential customers (leads) throughout the sales funnel, ultimately aiming to convert them into paying customers. This project provides a web-based solution for managing leads efficiently, tracking their status, and keeping all sales-related information organized.

## Features

- Add, update, and delete leads
- Track lead status and progress
- Manage follow-ups and actions
- User authentication and role-based access
- Dashboard with insights and statistics
- REST API for integration with other systems

## Tech Stack

- **Backend:** Django, Django REST Framework
- **Database:** PostgreSQL (or SQLite for local testing)
- **Cache / Queue:** Redis
- **Containerization:** Docker, Docker Compose
- **Frontend:** Django Templates (Bootstrap)
